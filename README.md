# PersonalBlogApp

A polished SwiftUI personal blog app designed for an iOS developer portfolio.

## Features

- Modern SwiftUI interface with portfolio-style homepage
- Article feed with tags, reading time, and featured content
- Rich article detail screen with structured sections
- About page highlighting skills, goals, and background
- Clean MVVM-style structure with reusable components

## Project Structure

- `PersonalBlogApp.xcodeproj`: Xcode project
- `PersonalBlogApp/`: SwiftUI source code and resources

## Run

1. Open `PersonalBlogApp.xcodeproj` in Xcode.
2. Choose an iPhone simulator.
3. Build and run.

## Notes

This project ships with mock blog content so it looks complete as a portfolio demo from day one.
